Welcome to the HTML Guide

Learn the basics of HTML, the standard markup language for creating web pages.

Introduction

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page and its content using a variety of tags and attributes.

This guide will walk you through the fundamentals of HTML, including basic structure, elements, attributes, and more.

Basic Structure

An HTML document consists of several elements, including:

  • <!DOCTYPE html>: Declares the document type and version of HTML.
  • <html>: Represents the root of an HTML document.
  • <head>: Contains meta-information about the document.
  • <title>: Sets the title of the document, displayed in the browser's title bar or tab.
  • <body>: Contains the content of the document, such as text, images, links, and more.

Here's a basic example of an HTML document structure:


      <!DOCTYPE html>
      <html>
        <head>
          <title>My First HTML Page</title>
        </head>
        <body>
          <h1>Hello, World!</h1>
          <p>This is a paragraph.</p>
        </body>
      </html>

Elements

HTML elements are the building blocks of a web page. Each element represents a specific type of content or functionality.

Some common HTML elements include:

  • <div>: Defines a division or section of a document.
  • <p>: Represents a paragraph of text.
  • <h1> to <h6>: Heading elements, with <h1> being the highest level and <h6> being the lowest.
  • <a>: Creates a hyperlink to another web page or resource.
  • <img>: Embeds an image into the document.
  • <ul> and <ol>: Represents unordered and ordered lists, respectively.
  • <li>: Defines a list item within a list.

Each HTML element may have one or more attributes that provide additional information or modify its behavior.

Attributes

HTML attributes provide additional information about an element or modify its behavior. They are specified within the opening tag of an element and usually come in name-value pairs.

Some common attributes include:

  • id: Specifies a unique identifier for the element.
  • class: Assigns one or more CSS classes to the element for styling purposes.
  • src: Specifies the URL of the resource, such as an image or script.
  • href: Specifies the URL of the hyperlink target.
  • alt: Provides alternative text for an image.
  • title: Provides additional information about the element, typically displayed as a tooltip.

Attributes can vary depending on the element they are applied to, and some elements may have specific attributes unique to their purpose.

CSS Guide

CSS (Cascading Style Sheets) is used to style the appearance of HTML elements on a web page.

Here's an example of styling an HTML element:


      /* CSS code */
      .my-element {
        color: blue;
        font-size: 16px;
      }

In the above example, we've defined a CSS class named my-element and applied styles to it. This class can then be used in HTML elements by adding the class attribute.

JavaScript Guide

JavaScript is a programming language that allows you to add interactivity and dynamic behavior to your web pages.

Here's an example of using JavaScript to display an alert:


      // JavaScript code
      alert('Hello, World!');

In the above example, the alert() function displays a dialog box with the message "Hello, World!". This is a simple example of JavaScript's ability to interact with users.
